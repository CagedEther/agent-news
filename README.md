# agent-news

Weekly notes on AI agents, coding harnesses, and related threads and videos.

Current digest: [`agent-news-may6.md`](./agent-news-may6.md).

## Publish to GitHub

From this directory, after [installing](https://cli.github.com/) and signing in (`gh auth login` or set `GH_TOKEN`):

```bash
gh repo create agent-news --public --source=. --remote=origin --push
```

That creates a public repo named `agent-news` under your account and pushes `main`.
